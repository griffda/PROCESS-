! Outputs:

* scalars
	geom%ip --> this is an output if q95 was an input
	mhd%ip --> plasma current in MA

	geom%q95 --> this is an output if ip was an input
	mhd%q --> q95

	comp%che --> Helium concentration (computed if globtau(3) is finite)
	comp%cxe --> Xenon concentration (computed if comp%psepplh_sup finite)
	comp%car --> Argon concentration (computed if comp%qdivt finite)
	

	ped%teped --> computed if num%ipedestal=2
	ped%nped --> computed according to inp0%fgw
	ped%nsep --> computed according to inp0%fgws

	
	mhd%equilcheck = 0. if equilibrium crashed, 1 if ok
	mhd%betan : total beta_N (thermal + alphas)
	
	loss%pfus : fusion power in MW (5* alpha power)	
	loss%taueff : confinement time taue in s
	loss%H : H factor
				tau_scal = loss%taueff/loss%H
	loss%Wth : plasma thermal energy in MJ
	loss%prad: total radiated power (core + edge) in MW
	loss%pradedge: edge radiated power in MW
	loss%pradcore: core radiated power in MW
	loss%psync : total synchrotron power (MW)
	loss%pbrehms : brehmstralung power MW
	loss%pline: line radiation MW
	loss%psepi: ion separatrix power MW
	loss%piepv: volume averaged equipartition power MW/m^3
	loss%peaux: effective auxiliary electron heating in MW
	loss%piaux: effective auxiliary ion heating in MW
	
	loss%pdiv : divertor power in MW/m^2

	loss%Psep: separatrix power	MW
	loss%PLH: LH transition power  MW

	loss%dfuelreq : requested fueling in p/s for D
	loss%tfuelreq : requested fueling in p/s for T

	loss%hepumpreq = He production globally in p/s (to be pumped)
	
	mhd%vp : plasma volume in m^3
	mhd%q_sep : separatrix q
	mhd%vloop : loop voltage in V
	mhd%fbs : bootstrap current fraction
	mhd%f_ni : total non inductive current fraction (CD + bs)


	i_flag = 1 if OK, -1 if not converged after itermax, -2 equilibrium crashed, 0 transport crashed



* profiles	
	radp%x : normalized coordinate r/a 
	radp%ne : electron density profile in 10^19 m^-3
	radp%te : electorn temperature profile in keV
	radp%ti : ion temperature profile in keV

	radp%qprof : safety factor profile
	radp%jpar : parallel current profile in MA/m^2
	radp%jbs :  bootstrap current profile
	radp%jcd : current drive profile
	radp%ipol : poloidal current R*BT profile in T*m
	radp%psi : poloidal flux profile in Wb
	radp%volum : plasma volum profile in m^3
	





! Suggested usage:

 - always set Psep_PLH inf and sup to like ~1 to ~1.1 or 1.2
	- qdivt = 0 to start
	- Hfac_inp = 0.95 or 1 for baseline scenario, can try 0 but transport model will change later on
	- q_control ~30 - 50 MW
	- pedestal model 2
	- equiltype 1
	- impmodel 0 or 1 doesnt matter
	- globtau(3) ~ 5 - 10
	- f_ni or v_loop should be dictated by discharge duration (comes from SYS code)
